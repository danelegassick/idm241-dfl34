Microinteractions IDM241

Alpha Information - Original Website Link

Triggers

  • Hover over tab
  • Click on tab

Rules

  • On hover, mouse changes to click state (hand icon)
  • On click, the tab's border bottom is changed from gray to black and text is changed to bold
  • On click of new tab, the old tab's border bottom is changed from black to gray, and the text is changed from bold to normal
  • On click, page data changes to display accurate information from the tab (i.e. Listings displays listings, but Searches will display search data respectively)

Feedback

  • On hover, mouse changes to click state (hand icon)

Loops and Modes

  • Mode: The bottom border and bold text will hold until the user has clicked a different tab
